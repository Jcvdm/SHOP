## KEY PLACEMENT RULES

1. **T&Cs Section Location**:
   - Assessment Report: After ASSESSMENT NOTES, before FOOTER
   - Estimate: After NOTES, before FOOTER
   - FRC Report: After line items/decisions, before FOOTER

2. **Styling**:
   - Blue header bar with white text
   - Light gray background box
   - 9pt font size
   - 1.5 line height
   - Bordered box with padding

3. **Visibility**:
   - Only shows if T&Cs text is configured in settings
   - Empty T&Cs = section not rendered
   - Multi-line formatting preserved

4. **Page Breaks**:
   - Use `page-break-inside: avoid` to keep T&Cs section together
   - Prevents T&Cs from splitting across pages
